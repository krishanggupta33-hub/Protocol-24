# Protocol-24
It is a controller that uses Wi-fi And infrared frequency to control the ac it is based on NodeMCU ESP8266 And IR receiver and blasters It makes any air conditioner smart

I named it protocol 24 because I have seen many Turn the temperature below 24 and The compressor of the air conditioner Keeps running continuously So that's I've made this Project so when someone turns down the temperature below 24 degree it starts a one hour timer Within itself And After one hour The temperature would again Be setted 24 degree Celsius. So the basic logic is when someone turns down the temperature Below 24 degrees Celsius It starts at Timer of one hour and after one hour It changes the temperature back to 24 degrees Celsius. This would keep the room Cool, It is good for the environment and Also the air conditioner will consume less electricity

It also have Website through which You can control the air conditioner And it is also connected To Google home So you can use your Google assistant to control the air conditioner

So basically I have used sinric pro To The microcontroller to the Internet. 

I've used AI To search about how can Connect my ESP82 With Internet and how can I use it with Google home.
